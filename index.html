Piano Unite

Piano Unite is a tool that enables teachers and students to work on elemental piano lessons. Teachers will be able to create customized exercises for their students, while students will learn as they play. Piano Unite is able to evaluate performance on each of the exercises and will reward the student's progress. In addition, the user will be able to import any MIDI file to use it as a new game level.

• Developed as a bachelor thesis at UCM.

• Built with Unity and C++.

• The project's memory for this bachelor thesis is not publicly available at the moment.

Motor Casa Paco
